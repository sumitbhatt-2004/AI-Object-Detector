**🚀 Real-Time AI Object Detector (YOLOv8)**

This project implements a real-time object detection system using the YOLOv8 (You Only Look Once) architecture. It is designed to identify and track multiple objects within a live video stream with high precision and low latency.

**Real-Time Analytics:** Processes live webcam feeds to identify objects instantly.

State-of-the-Art Model: Utilizes the YOLOv8 Nano model for an optimal balance between speed and accuracy.

Web-Based Interface: Built with Streamlit to provide a clean, accessible dashboard for users.

Cloud Compatibility: Integrated with Streamlit-WebRTC to support browser-based camera access on deployed links.

**📊 Business Applications & Use Cases**

Leveraging my background in Business Analytics, I have designed this project to address real-world industrial challenges:

Retail Intelligence: Analyzing customer footfall and heatmaps to optimize store layouts.

Operational Efficiency: Automating inventory counting and tracking in warehouse management.
